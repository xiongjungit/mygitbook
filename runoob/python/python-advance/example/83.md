#Python 练习实例83


题目：求0—7所能组成的奇数个数。
程序分析：无。

程序源代码：

```
#!/usr/bin/python
# -*- coding: UTF-8 -*-

if __name__ == '__main__':
    sum = 4
    s = 4
    for j in range(2,9):
        print sum
        if j <= 2:
            s *= 7
        else:
            s *= 8
        sum += s
    print 'sum = %d' % sum
```

以上实例输出结果为：

```
4
32
256
2048
16384
131072
1048576
sum = 8388608
```
